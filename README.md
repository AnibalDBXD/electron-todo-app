# Electron Todo-app with ReactJS

# Initialization

1. ``` npm install ```
2. ``` npm run dev ```

# Build

- Electron build ``` $ npm run electron-build```
- React build ``` $ npm run react-build```
- Installer ``` $ npm run build```

# Technologies

- Electron
- ReactJS
- Typescript
- ChakraUI
- Eslint (Airbnb)
